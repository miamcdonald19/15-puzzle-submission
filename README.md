# Sliding Puzzle (15 Puzzle / 8 Puzzle)

Foundations of Programming – Final Project

## Project Description

This project implements a graphical sliding puzzle game in Python using pygame.

The game includes both the classic **15-puzzle (4×4 board)** and the **8-puzzle (3×3 board)**.  
The goal is to arrange the numbered tiles in ascending order with the blank tile in the bottom-right corner by sliding tiles into the empty space.

Features of the final project include:

- 4×4 sliding puzzle (15-puzzle)
- 3×3 puzzle mode (8-puzzle)
- Guaranteed solvable puzzle generation
- Move counter
- Timer
- Optional diagonal movement mode
- Clickable graphical interface using pygame
- Unit test suite verifying puzzle behavior

---

## How to Run the Project

### Run the Game

1. Open a terminal.
2. Navigate to the project folder.
3. Run:

python game.py

This launches the pygame window where the puzzle can be played.

---

### Run the Test Suite

In the project folder, run:

python -m unittest -v test_puzzle.py

All tests should pass.

---

## Controls

Mouse:
- Click a tile next to the blank space to move it.

Keyboard:
- R → reshuffle the puzzle
- 3 → switch to 3×3 puzzle
- 4 → switch to 4×4 puzzle
- D → toggle diagonal movement mode

## Implementation Notes

The puzzle scramble is generated by starting from the solved board and applying a sequence of random valid moves.  
This guarantees that every scrambled puzzle is reachable and therefore solvable.

The puzzle logic is separated from the graphical interface so that the core functionality can be tested independently.
